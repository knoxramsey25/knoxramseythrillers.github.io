Glossary

Actuator
Device that changes physical conditions (pumps, valves, LEDs, fans) under control logic.
Advisory Mode
Cloud suggests setpoints; the edge executes only after local verification.
APTs (Advanced Persistent Threats)
Long-dwell intrusions by state-aligned or highly resourced actors.
Certificate Authority (CA)
Trust service that issues digital certs; if spoofed, attackers can pass as trusted.
Closed-Loop Control
Automatic adjustments based on feedback; powerful and risky without independent checks.
EC
Electrical Conductivity; proxy for nutrient strength in solution.
Edge
Local compute near sensors/actuators; reduces latency and preserves autonomy.
FarmCore
Edge-side automation: sensors, actuators, and a safety kernel enforcing local limits.
FarmLytics
Cloud analytics that ingests telemetry, trains models, and proposes “recipes.”
Furanocoumarins / Psoralens
Plant compounds that can become phototoxic upon UV exposure.
HMI
Human–Machine Interface displaying real-time status, alarms, and controls.
Immutable Log
Append-only record of events, signed to prevent tampering.
Model Weights
Learned parameters of an AI model; if altered, behavior changes even when code is unchanged.
PAL
Phenylalanine Ammonia-Lyase; enzyme initiating phenolic synthesis under stress.
PAR / PPFD
Light available to plants / intensity at canopy (photosynthetic photon flux density).
Photoreceptors
Plant proteins that detect spectra (phytochromes, cryptochromes, phototropins, UVR8).
ROS
Reactive Oxygen Species; short-lived oxidants that signal stress and damage cells at high levels.
Safety Kernel (v2)
Edge runtime validating recipes with dual-channel checks and simulated stress indices.
Shadow Test
Local simulation of a proposed recipe before real execution.
Setpoint
Target value for a controlled variable (e.g., pH 6.0, EC 2.2 mS/cm).
Spectral Ratio
Relative mix of light bands (blue/red/far-red/UV) delivered to canopy.
Telemetry
Time-series measurements streamed from sensors to controllers and analytics.
